Algorithm Information

Features

  • 6-way rotation capability for optimal fitting
  • Weight constraint validation
  • Dual algorithm approach (FFD and BFD)
  • Automatic optimization selection

Rotation Types

  • Type 0: Original orientation
  • Type 1: 90° rotation around X-axis
  • Type 2: 90° rotation around Z-axis
  • Type 3: 90° rotation around Y-axis
  • Type 4: 180° rotation around Y-axis
  • Type 5: 270° rotation around Y-axis

This 3D bin packing algorithm optimizes the placement of items into boxes while minimizing the number of boxes used. The algorithm tries all possible rotations of items to find the best fit and uses advanced heuristics to optimize packing efficiency.
